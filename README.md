# Exp - 3 RAILWAY MANAGEMENT SYSTEM

# AIM:

To study the problem statement, SRS document and draw all the UML diagrams of a Library Management System.

# SRS (Procedure):
Study and understand the requirements of the library system
Identify the users or actors who interact with the system like librarian, member, admin
List the main functions of the system such as issue book, return book, search book, add book
Identify main objects or classes like Book, Member, Librarian, Transaction, Catalog
Define relationships and interactions among users and system components
Create the workflow or process flow for activities like book issue and return
Organize the system structure and represent it using UML diagrams
